# Reglas de Amazon Q Developer - Proyecto React

Este directorio contiene las reglas de revisión de código para Amazon Q Developer.

## Archivos de reglas:

- `components.md` - Convenciones de componentes React
- `hooks.md` - Reglas de React Hooks
- `state.md` - Gestión de estado
- `styles.md` - Estilos y UI
- `performance.md` - Optimizaciones de rendimiento
- `accesibility.md` - Accesibilidad web (a11y)
- `testing.md` - Pruebas y testing
- `clean-code.md` - Convenciones de código limpio
- `security.md` - Seguridad
- `dependencies.md` - Gestión de dependencias
- `git.md` - Convenciones de Git
- `revision.md` - Guías de revisión de código
- `priorities.md` - Priorización de sugerencias

## Mantenimiento:

Para actualizar reglas, edita el archivo correspondiente y haz commit de los cambios. 
Amazon Q Developer detectará automáticamente las actualizaciones.
